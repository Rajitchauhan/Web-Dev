In JavaScript, getters and setters are special methods used for accessing and modifying the values of an object's properties.
They allow you to control how properties are read and written,
providing a level of encapsulation and enabling additional logic during property access
and modification.



Getter: A getter is a method that allows you to retrieve the value of a property.
It is defined using the get keyword followed by the property name.



Setter: A setter is a method that allows you to modify the value of a property.
It is defined using the set keyword followed by the property name.
