Call()

1. call() The call() method is used to invoke a function with a specified this value and individual arguments.

2. apply() The apply() method is similar to call(),
but it takes an array or an array-like object as its second parameter to pass arguments to the function.

3. bind() The bind() method creates a new function with the same body as the original
function but a fixed this value.

call(): Invokes a function with a specified this value and individual arguments.

apply(): Invokes a function with a specified this value and an array or array-like object as arguments.
bind(): Creates a new function with a fixed this value.
